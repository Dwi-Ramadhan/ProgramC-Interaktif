Deskripsi Program:
 Program ini saya buat menggunakan bahasa pemrograman C++
 dan DevC++ IDE. 
 Algoritma program ini dapat dilihat di file 'flowchart.jpg'.
 Saya berusaha membuat User Interface yang fleksible dan
 interaktif meskipun ini program command-line.
 Saya menggunakan fungsi system('cls') pada program ini,
 sehingga mungkin tidak  bekerja dengan baik dengan 
 compiler versi tertentu (yg tidak menyertakan library
 untuk fungsi system() )

Fitur Program :
1. Program Array
  - Menginput/Mengubah data di array
  - Mengurutkan data di array baik dari terkecil(ascend_sort)
    maupun dari terbesar(descend_sort)
  - Mencari data tertentu didalam array, bila ditemukan
    akan menampilkan indeks dimana data itu berada. Hal ini
    bergantung pada fungsi sort yg terakhir kali dijalankan.
2. Program menghitung nilai
  - Menginput nilai-nilai dan nama
     (input nama boleh mengandung spasi karena saya menggunkan fungsi getline())
  - Menghitung nilai akhir berdasarkan formula yg ditentukan
  - Menentukan predikat nilai akhir berdasarkan ketentuan
    yang telah ditetapkan

Cara Menjalankan:
 Untuk menjalankan, dgn membuka file 'a.cpp' di DevC++
 dan tekan tombol 'Compile & Run'
